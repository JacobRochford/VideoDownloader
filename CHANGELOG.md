# Changelog

All notable changes to this project will be documented in this file.

## 1.1.0 - 2026-04-18

- Added playlist detection and full playlist download support.
- Downloads are now saved in the `videos/` directory, with playlists organized into separate subfolders.

## 1.0.0 - 2026-04-18

- `Install-Dependencies.bat` for automated setup of required tools.
- `VideoDownloader.bat` for downloading videos using yt-dlp and FFmpeg.
- Dependency checks and clearer error messaging.
